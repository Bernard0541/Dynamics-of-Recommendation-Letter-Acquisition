# Dynamics-of-Recommendation-Letter-Acquisition
A deterministic and stochastic mathematical modeling framework for studying recommendation letter acquisition, institutional discouragement, lecturer availability, and student success using ordinary and stochastic differential equations, Ornstein–Uhlenbeck processes, sensitivity analysis, and survival probability.

This repository contains the MATLAB implementation accompanying the study

**Dynamics of Recommendation Letter Acquisition: A Deterministic and Stochastic Modeling Approach**

The project develops the first mathematical framework for analyzing the acquisition of recommendation letters in higher education by combining deterministic and stochastic compartmental models. The model investigates how lecturer availability, recommendation rates, institutional discouragement, and peer influence affect students' ability to obtain recommendation letters.

## Features

- Deterministic recommendation letter model
- Stochastic model using Ornstein–Uhlenbeck processes
- Institutional Discouragement Index
- Stability analysis
- Forward bifurcation analysis
- Local sensitivity analysis
- Latin Hypercube Sampling (LHS)
- Partial Rank Correlation Coefficient (PRCC)
- Monte Carlo simulations
- Discouragement survival probability
- Numerical simulations and visualization

## Model Components
### Student Compartments
- Susceptible students
- Informed students
- Connected students
- Students receiving recommendation letters
- Failed students

### Lecturer Compartments
- Recruited lecturers
- Available lecturers
- Unavailable lecturers
- Negative-influencer lecturers
